# Readme-SPOUserRemover

## Overview
This script is designed to efficiently remove users from SharePoint Online, with comprehensive support for various configurations and multi-site management.

## Features
- **Multi-Site Support:** This script can operate across multiple SharePoint sites using a single configuration, making it an ideal solution for organizations with extensive SharePoint environments.
- **User Information List Removal:** The script allows administrators to manage User Information Lists effectively by identifying users for removal and ensuring that no residual data is left behind.
- **Throttle Handling Improvements:** Users can customize throttle settings to minimize the impact on SharePoint's performance. Adjustable parameters ensure smooth operations even during peak hours.

## Configuration Options
### Required Parameters
1. **Site URLs:** List of SharePoint site URLs where users need to be removed.
2. **Usernames:** The usernames or email addresses of the users to be removed.

### Optional Parameters
- **Throttle Limits:** Specify the maximum number of requests per second to stay within SharePoint's API limits.
- **Logging Level:** Choose the verbosity of the logs generated by the script (e.g., Info, Warning, Error).

### Usage
To execute the script, run the following command:
```
python SPOUserRemover.py -u <usernames> -s <site-urls> [options]
```

Ensure you update the configurations according to your organizational needs.

## Conclusion
This script offers a powerful way to manage user removals in SharePoint Online, with detailed configurations to cater to diverse setups and use cases.
